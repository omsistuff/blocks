Blocks

Blocks is a css framework that uses lots of small blocks that can be used inside each other to form rich and personalized interfaces.

Style is strongly inspired from Google Material Design, and incorporate the 3 main principles

Material

the design is built on a system of layers, which stack on top of each other without ever overlapping

Bold

The elements are large, graphic and are identifiable at first glance.

Motion

The elements appear, move and disappear in a natural way, so as not to lose the user.
